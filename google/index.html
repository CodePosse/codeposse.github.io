Prorotypes demos Tim Hunold built

Some prototypes I made at Google but I have them very limited for your preview. These are Angular 8 and Dart. I can "show my work" for a couple items, but you know how NDAs work. I built an Angular 8 boilerplate for the Google Ads UX research team to use to build demos with copy-paste. [link]

  • Ad-Planner
    This is a demo of selected search phrases that will auto-complete (best foundation, cookware, couch, credit card, employee survey, harley davidson, invoice template, luggage, running shoes, stem cell therapy, or vacuum pumps). Once a phrase is used, go to the "Refine Keywords" right nave and start seeing inclusive, and exclusion filters along with changes in the results. When is the last time you saw a search that was "must include", "may include" or "must NOT include" as options! It's all in the right rail.
  • Forecast
    This is more about a re-arranged right navbar, a few items expand but it is really actualized in the Sliders below.
  • Performance Planner
    This is a customized scatter plot that has tooltips, crosshairs, colors, (all of this in one graph is actually a big deal, I had to create it from scratch and add to the Google internal repos!) and is manipulatable from the dropdowns above the table. Right rail nav kinda functions.
  • Sliders
    This is a forecaster for campaign spend. Another big deal for Google. You can chanmge values in "Conversions", "CPA" then ths slider below move and are movable. The chart changes values AND you can change some values in the table below "Budget", "Conversions", "Average CPA" and you can click the dots/pencil to see a detailed view in a push-out drawer. There is a LOT going on here, it is a very big deal for google, so I am pretty proud.
